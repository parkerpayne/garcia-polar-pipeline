Setup (DEPRICATED)

This page will walk you though the entire process to set up a new computer. The downloads should all be linked on this page, so no need to go find stuff yourself. The pipeline is designed for machines running Ubuntu, and I am relatively confident that it will not work on Windows.

1 - Pipeline Files

The files for the pipeline itself are here. Unzip the contents into the Documents folder.

2 - Minimap

Minimap can be found here. The default configuration (which can be changed) is to unzip into the home directory.

3 - Samtools

The samtools download is here. Similar to minimap, unzip the samtools into the home directory.

4 - Nextflow

Install nextflow using the following commands (can be run anywhere):

curl -s https://get.nextflow.io | bash
sudo mv nextflow /usr/local/bin

5 - Vep

Download the vep zip here. Both folders will be placed into the home directory.

7 - Merge

Download the merge files here. All the python files will be placed into the root of the Documents directory, same place as the pipeline python file itself.

8 - Bedtools

Install bedtools using the command:

sudo apt install bedtools
